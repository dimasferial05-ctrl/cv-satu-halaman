# CV Satu Halaman - Dimas Ferial Hidayat

Ini adalah proyek pembuatan CV satu halaman yang responsif dan minimalis. Proyek ini dibuat untuk mempraktikkan dasar-dasar pengembangan web (_frontend_).

## 🔗 Live Demo

Kamu bisa melihat hasil akhir dari CV ini melalui tautan berikut:
**[Lihat CV Live di sini](https://dimasferial05-ctrl.github.io/cv-satu-halaman/)**

## 💻 Teknologi yang Digunakan

- HTML5
- CSS3

## 🚀 Cara Menjalankan Proyek Secara Lokal

Jika ingin melihat atau memodifikasi kode ini di komputermu:

   ```bash
   git clone [https://github.com/dimasferial05-ctrl/cv-satu-halaman.git](https://github.com/dimasferial05-ctrl/cv-satu-halaman.git)
   ```
